FALL(IL)

The game tells about the life of an outcast at school. You need to get the bullies to get behind you. The game will have many endings, so there are a lot of options for solving the problem.

Fall(IL) is a game set in the danganronpa universe. For the most part, this is fanfiction for a friend's birthday. But even so, I tried so I think I should share this with the world. If someone is interested in this visual novel, then I will be glad.

The game will release on November 6, 2023. Don't take it seriously, it's just fanfiction
